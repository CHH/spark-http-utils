# Spark HttpUtils

_Utilities for composing HttpKernels_

## Install

Get [composer]:

    $ wget http://getcomposer.org/composer.phar

Install `spark/http-utils` package:

    $ php composer.phar require spark/http-utils:*@dev

[composer]: http://getcomposer.org
